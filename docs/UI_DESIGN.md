# UI / Product Design (MVP)

## Main screens
1. **Dashboard**
   - KPIs: appointments today, open work orders, revenue today.
2. **Calendar**
   - Daily/weekly board by bay or mechanic.
3. **New Appointment Modal**
   - Customer, vehicle, service reason, time slot.
4. **Service Order Detail**
   - Diagnosis, parts/labor, evidence, timeline.
5. **Quote Approval View (client)**
   - Simple mobile-first view: approve/reject.
6. **Checkout / Payment**
   - Register payment and close order.

## Visual design system
- Typography: Inter (or system font fallback).
- Color roles:
  - Primary: #2563eb
  - Success: #16a34a
  - Warning: #d97706
  - Danger: #dc2626
- Spacing scale: 4 / 8 / 12 / 16 / 24 / 32.
- Radius: 10px cards, 8px controls.

## UX principles
- Most common actions in max 3 clicks.
- Clear status chips on every card.
- Form validation inline + friendly errors.
- Responsive first for advisor mobile workflow.
